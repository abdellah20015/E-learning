Cours 4 : Les bases du MySQL - Résumé

Ce cours vous apprend les bases du MySQL, un système de gestion de bases de données relationnelles. Vous allez apprendre à :
  • Installer et configurer MySQL sur votre ordinateur
  • Créer et supprimer des bases de données et des tables
  • Insérer, modifier, supprimer et sélectionner des données
  • Utiliser les clés primaires, les clés étrangères et les contraintes d'intégrité
  • Utiliser les opérateurs, les fonctions et les clauses SQL
  • Joindre des tables avec les jointures internes et externes
  • Utiliser les sous-requêtes et les vues
  • Utiliser les procédures stockées, les fonctions et les déclencheurs

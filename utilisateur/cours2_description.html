Cours 2 : Les bases du HTML et du CSS - Description de vidéo

Dans cette vidéo, vous allez découvrir les bases du HTML et du CSS, les langages indispensables pour créer des pages web. Vous allez voir comment utiliser un éditeur de code et un navigateur web pour créer et afficher votre première page HTML. Vous allez ensuite apprendre à utiliser les balises HTML les plus courantes pour structurer et enrichir votre contenu. Vous allez également apprendre à mettre en forme le texte et les éléments HTML avec le CSS, en utilisant des sélecteurs, des propriétés et des valeurs. Vous allez enfin apprendre à modifier la disposition, la position et l'alignement des éléments HTML avec le CSS, en utilisant des propriétés comme display, position, float et flexbox.

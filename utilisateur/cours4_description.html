Cours 4 : Les bases du MySQL - Description de vidéo

Dans cette vidéo, vous allez découvrir les bases du MySQL, un système de gestion de bases de données relationnelles très populaire et utilisé dans de nombreux projets web. Vous allez voir comment installer et configurer MySQL sur votre ordinateur, et comment utiliser un logiciel client pour vous connecter à votre serveur MySQL. Vous allez ensuite apprendre à créer et supprimer des bases de données et des tables, et à manipuler des données avec le langage SQL. Vous allez apprendre à utiliser les clés primaires, les clés étrangères et les contraintes d'intégrité pour assurer la cohérence et la sécurité de vos données. Vous allez également apprendre à utiliser les opérateurs, les fonctions et les clauses SQL pour effectuer des opérations complexes sur vos données, comme les trier, les filtrer, les regrouper ou les agréger. Vous allez aussi apprendre à joindre des tables avec les jointures internes et externes, pour combiner des données provenant de sources différentes. Vous allez enfin apprendre à utiliser les sous-requêtes et les vues, pour simplifier vos requêtes SQL, et à utiliser les procédures stockées, les fonctions et les déclencheurs, pour automatiser des tâches répétitives ou complexes.

Cours 2 : Les bases du HTML et du CSS - Résumé

Ce cours vous apprend les bases du HTML et du CSS, les langages essentiels pour créer des pages web. Vous allez apprendre à :
  • Créer la structure et le contenu d'une page HTML
  • Utiliser les balises HTML les plus courantes
  • Insérer des images, des liens, des tableaux et des formulaires
  • Mettre en forme le texte et les éléments HTML avec le CSS
  • Utiliser les sélecteurs, les propriétés et les valeurs CSS
  • Appliquer des styles à la page entière ou à des parties spécifiques
  • Modifier la disposition, la position et l'alignement des éléments HTML
